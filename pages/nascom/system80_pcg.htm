Nascom System 80 - Programmable Character Generator

Description

Before Nascom's financial problems, they where designing a new system referred to as System 80.  This got as far as the design stage and adverts where placed with magazines but production never took place.

Designed to take over the standard character generator to give a resolution of 384 x 256 using 2K of onboard static RAM (2114's).  The graphics ROM was to be moved from the Nascom 2 to this board. Never reached production.

Technical Details

  • 2K RAM
  • 384 x 256 Graphics
