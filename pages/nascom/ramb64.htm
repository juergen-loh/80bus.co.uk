RAM 'B/64' - 64K DRAM Memory Card

Description

Why the original RAM 'B' card was only equipped with 48K is something of a mystery.  Gemini produced a 64K card as one their first products which was fully compatible with page mode.  Being a similar price it was fairly obvious which would be the card of choice.

Eventually a professionally produced 16K upgrade add on was produced to give a full 64K.  Little information exists as to who produced this device but the first one I encountered was in a factory built Nascom 3.  This suggests it was an officially approved product upgrade.

Technical Details

  • DRAM Memory Card
  • 64K via 32 x 4116 Devices (With piggy back board)
  • Page mode support
