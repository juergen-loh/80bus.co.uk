Nascom 2 Keyboard

Description

The typewriter type keyboard supplied with the Nascom 2.  This was rather unusual at the time being a full 'normal' type keyboard, rather than the calculator types used by other machines.  It was an extension of the Nascom 1 keyboard and featured cursor control keys to allow on screen editing via the new NAS-SYS 1 monitor program.

Technical Details

  • Full travel keyboard
  • Enhanced Nascom 1 keyboard
  • Rather nice to use actually!
