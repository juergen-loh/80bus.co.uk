Lucas / Nascom - Full Height Drives

Description

The first disk drives introduced by Lucas where full height, single sided, quad density units giving approximately 720K per disk.  The drives are re-badged TEAC units.  Two systems where offered, one using NAS-DOS to be compatible with existing software and one using CP/M 2.2. 

Technical Details

  • TEAC Drives
  • Single Sided, 80 Track Units
  • Full Height
  • Very heavy!
