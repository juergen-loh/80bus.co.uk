Lucas / Nascom - Half Height Drives

Description

The full height units where eventually replaced by half height TEAC drives.  These where a bit lighter, double sided if required, and had a much higher stepping rate.  To take advantage of this speed, later versions of NAS-DOS had modified stepping rates, which caused a few issues using later software with early drives.

Technical Details

  • TEAC Drives
  • Single Sided and Double Sided, 80 Track Units
  • Half Height
  • Heavy!
