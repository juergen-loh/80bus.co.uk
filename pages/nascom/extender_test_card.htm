Nascom Extender Test Card

Description

This is the rarest standard Nascom item by far.  It is a test card designed to help diagnose faults on Nascom systems and was not normally available to end users.  It comprises displays for address and data bus values, a probe and an 80-Bus edge connector.  The board also has a smaller connector which looks like it would fit the Nascom 1 edge connector, however this is not certain.  Only two are known to still exist.

Technical Details

  • 80-Bus and Nascom 1 Connectors
  • Displays for Address and Data Values
  • Reset Button
  • Single Step (?) Button
