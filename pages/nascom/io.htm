Nascom I/O Card

Description

General I/O card containing 3 PIO's, 1 CTC and 1 UART. Originally announced as part of System 80.This card was the only System 80 card  to make it into production following the takeover from Lucas Logic.

Technical Details

  • 3 x Z80A - 4MHz
  • 1 x UART
  • 1 x CTC
