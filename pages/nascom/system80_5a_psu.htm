Nascom System 80 - 5A PSU

Description

Before Nascom's financial problems, they where designing a new system referred to as System 80.  This got as far as the design stage and adverts where placed with magazines but production never took place.

A 5A power supply was announced to go with the System 80 components. It is presumed that this would have been designed to fit into the standard System 80 enclosure.
