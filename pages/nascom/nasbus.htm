Nascom - NAS-BUS

Description

This is where the whole 80-BUS world started.  The bus interface for the Nascom 2 was published as a standard to anyone who wanted to build new cards.  This allowed a flourishing industry to be created which led to approximately one hundred compatible cards to be produced from a variety of manufacturers.

Technical Details

  • Fully Buffered Bus
  • 4MHz Operation
  • Built from Cheap / Standard Components
  • Standardized
