Nascom System 80 - Colour Encoder Board

Description

Before Nascom's financial problems, they where designing a new system referred to as System 80.  This got as far as the design stage and adverts where placed with magazines but production never took place.

Designed to produce a PAL, SECAM, NTSC or RGB output in a choice of 16 colours, split between foreground and background.  The board was meant to be used in conjunction with the Programmable Character Generator board above. Colour definition was on a 96 x 48 grid produced using 6K of 4118 memory. Never reached production. The board was capable of producing 256 colours but the method is unclear.

Technical Details

  • Used with PCG
  • 96 x 48 Colour Grid

Documentation
