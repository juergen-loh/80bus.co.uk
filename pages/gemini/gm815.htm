GM815 - Dual Floppy Drives

Description

This is  the first stand floppy drive unit supplied by Gemini.  These are the same drives as supplied for the Henelec system (GM805) but without the built in 1771 based disk controller.  In this instance, the GM809 FDC card is used as the controller. 

Technical Details

  • Pertec FD250 drives, 40 track, double sided, single or double density
  • Maximum of 350K per drive

Documentation

  • Required
