GM863 - Byte wide Static RAM Card

Description

This card  is designed to hold up to eight byte wide static RAM devices.  These are arranged in two addressable pairs allowing them to be placed anywhere in the memory map. A battery backup system is available to preserve the memory contents.

Another version of this card exists (GM853), using byte wide EPROMS. 

Technical Details

  • Supports up to 8 * 8K RAM's
  • Addressed as either paged memory 4 * 64K, extended addressing or as a flat memory for 16 bit systems
