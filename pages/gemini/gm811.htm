GM811 - Z80A CPU card

Description

This processor card was developed to give Gemini the capability to market a complete computer. Until this stage they had been an add-on only company. Shipped with RP/M, a CP/M ROM based look alike monitor. This allows CP/M software to be executed without disks.

Technical Details

  • Z80A CPU
  • Keyboard interface
  • Able to boot CP/M
