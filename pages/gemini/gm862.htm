GM862 - 256K DRAM Card

Description

This card  is designed to deliver the larger memory required for later 8 bit and early 16 bit systems.  It uses the extended address mode for  addressing 512K or 1024K as defined by Gemini.  This is useful for CP/M 3 paging, and for 16 bit systems such as the 8088 / 8086 cards. 

Technical Details

  • 256K Dynamic RAM
  • Addressed as either paged memory 4*64K, extended addressing or as a flat memory for 16 bit systems
  • Up to four cards could be used giving 1024K
