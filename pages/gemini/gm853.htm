GM853 - Byte wide EPROM Card

Description

This card  is designed to hold up to eight byte wide EPROM devices.  These are arranged in two addressable pairs allowing them to be placed anywhere in the memory map. 

Another version of this card exists (GM863), using byte wide static RAMS.  A battery backup system is available to preserve the memory contents.

Technical Details

  • Supports up to 8 * 27256 EPROM's
  • Addressed as either paged memory 4 * 64K, extended addressing or as a flat memory for 16 bit systems
