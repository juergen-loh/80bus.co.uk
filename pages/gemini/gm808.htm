GM808 - EPROM Programmer

Description

This EPROM programmer plugs into the parallel port of a Nascom 1 or 2. Used to program 2708's and 2716's. The circuit layout was changed between the original Bits & PC's version and the Gemini version.

Technical Details

  • Attaches to parallel port
  • Programs 2708 and 2716 EPROM'S
