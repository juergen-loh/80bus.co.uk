GM802 - 64K Dynamic RAM Card

Description

This was the first board for the 80-BUS from Gemini.  It was the first full 64K RAM card available and supported the Nascom page mode memory paging system. This became the board of choice over the Nascom RAM B which only supported 48K.  The full 64K became a virtual essential for CP/M systems.

Technical Details

  • 64K Dynamic RAM
  • Uses 4116 DRAM
  • Support Nascom page mode
