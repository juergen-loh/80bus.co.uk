GM805 - Henelec Disk System

Description

This is the original disk system for Nascom's. It was designed to be attached to the PIO and could be used with a Nascom 1 or 2.  Originally supplied with CP/M 1.4 and later CP/M 2.1.  A disk 'tape recorder' was also available via a simple application called D-DOS.

Technical Details

  • Supports up to 3 drives
  • Based on small 1771 controller card
  • Supports single density only (2 sides, 35 tracks, 128 bytes / sector, 18 sectors)
