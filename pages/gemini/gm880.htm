GM880 - 64180 CPU / RAM Card

Description

This is the ultimate Z80 based CPU card developed for 80-BUS Systems.  The 64180 processor is a 16 bit super-Z80, clocked at 12MHz.  The card also supports 256K of RAM addressable via the Gemini memory paging scheme. 

The card is completely backwards compatible with the GM813 CPU card and could be used as a drop in replacement to turbo charge a system. 

This card is extremely rare.

Technical Details

  • 256K Memory
  • Hitachi 64180 16 bit CPU
