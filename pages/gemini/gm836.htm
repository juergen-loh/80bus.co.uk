GM836 - MultiNet Interface Board

Description

This board is designed to connect to the PIO on a Gemini CPU card.  It allows access to the Gemini Network (MultiNet) via a high speed serial connection run over coaxial cable.  The software used is CP/NET which allows CP/M based diskless workstations to be attached to a file server.

In an early attempt to hide technical details, the device numbers have been erased from the serial I/O chip.
