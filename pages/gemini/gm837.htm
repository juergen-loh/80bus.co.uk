GM837 - Climax Colour Graphics Card

Description

This card was originally developed by Climax Computers before being taken over by Gemini.  It has a 256 x 256 x 16 colour display driven by a EF9365 device.  Video data is held in 32K of DRAM and the output is either via PAL for TV display output or via RGB output.

Technical Details

  • Resolution of 256 x 256
  • 16 Colours available
  • Thompson EF9365 graphics display processor
