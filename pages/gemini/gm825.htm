GM825 - Dual Floppy Drives

Description

This is  the second generation floppy drive unit supplied by Gemini.  It comprises of a single / twin 5.25 disk unit using Micropolis drives and a GM804 PSU.  For use with the GM809 / GM829.  Capacity of 400K single sided / 800K double sided.

Technical Details

  • Micropolis single dual sided, quad density drives
  • Maximum of 400K per side, 800K per drive

Documentation
