GM832 - 80 x 25  Super Video Controller Card

Description

The original Intelligent Video Controller was superseded by the Super Video Controller.  The improved video board to replace the GM812 with a faster CPU (6MHz Z80B) and revised 40*25 screen format amongst other items.  It also has a limited graphics capability.  It is odd to note that this display card had a more powerful CPU than the main computer.

Technical Details

  • Z80B CPU with 2K useable RAM
  • Keyboard interface
  • Limited graphics capability 
  • I/O driven so full 64K memory map available for CP/M
