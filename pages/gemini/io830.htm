IO830 - PLUTO Mini Palette Board

Technical Details

This  is a piggy back board for the IO828 PLUTO board.  It allows any 8 colours to be selected from a choice of 4096. The image below is a PLUTO with Mini Palette attached.

Technical Details

  • Allows 8 colours from a palette of 4096
