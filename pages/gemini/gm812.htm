GM812 - 80 x 25  Video Controller Card

Description

The intelligent Video Controller was designed to display 80 * 25 text. Control of the card was done via an board Z80A. This card was the first way to get normal video for 80-BUS systems which is essential for running most CP/M applications.  This card with the GM811 CPU, GM809 FDC and GM802 64K RAM together would give a fully functional CP/M system.

Technical Details

  • Z80A CPU with 2K useable RAM
  • Keyboard interface
  • Limited graphics capability 
  • I/O driven so full 64K memory map available for CP/M
