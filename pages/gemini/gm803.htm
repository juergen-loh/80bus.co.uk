GM803 - 16K / 32K EPROM Card

Description

This card supports up to 16 EPROM devices of type 2708 and / or 2716, giving a maximum storage of 32K. It also supports the Nascom page mode memory paging system.  A socket is also available for the 8K Nascom Basic ROM.  Basic can be run with wait states while the rest of the board operates at a full 4MHz

Technical Details

  • 16 EPROM sockets supporting a mix of 2708's and 2716's
  • Support Nascom page mode
  • Socket for Nascom 8K Basic ROM
