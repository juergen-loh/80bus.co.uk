MAP-80 MPI Card

Description

This card is designed to cram as much I/O functionality as possible onto one device.   It contains support for a serial I/O device, a CTC and a floppy controller.  The controller could be configured to emulate either a Lucas / Nascom card or a Gemini Card.  When emulating a Nascom card, it could also support a SASI controller interface giving Lucas a method of deploying hard disk solutions.

Technical Details

  • Serial Interface
  • CTC
  • Configurable FDC Interface
