MAP-80 Video Floppy Controller Card

Description

This interesting card is designed to allow a user to build a system down to a price.  This card has a GM809 compatible FDC and an 80*25 video card.  The card was designed to be as compatible as possible with the Gemini IVC as possible but this was limited by the fact that no onboard processor was used.  It is thought that most of the work was done via software.

Technical Details

  • GM809 Compatible floppy controller
  • 'Work Alike' video controller based around the Gemini IVC
